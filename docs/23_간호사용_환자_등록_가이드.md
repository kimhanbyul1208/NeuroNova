## 개요

본 가이드는 NeuroNova 시스템에서 간호사가 내원 환자를 등록하는 절차를 안내합니다.

### 주요 기능
- 환자 기본 정보 등록
- 담당 의사 배정
- 임시 비밀번호 자동 생성
- 환자 정보 출력 및 전달

---

## 접속 방법

### 1. 로그인
1. React Web 어플리케이션 접속: `http://[서버주소]:5173`
2. 간호사 계정으로 로그인
   - 아이디: 병원에서 발급받은 계정
   - 비밀번호: 초기 비밀번호 (첫 로그인 시 변경 필요)

### 2. 환자 등록 페이지 이동
- 로그인 후 상단 메뉴에서 **"환자 등록"** 선택
- 또는 직접 URL 접속: `http://[서버주소]:5173/admin/patient-registration`

---

## 환자 등록 절차

### Step 1: 환자 기본 정보 입력

#### 필수 정보
1. **성명**
   - 성 (Last Name): 한글 또는 영문
   - 이름 (First Name): 한글 또는 영문

2. **주민등록번호**
   - 형식: `YYMMDD-GXXXXXX` (하이픈 포함 13자리)
   - 예시: `901215-1234567`
   - ⚠️ **중요**: 주민번호는 암호화되어 저장됩니다

3. **생년월일**
   - 형식: `YYYY-MM-DD`
   - 달력 선택 또는 직접 입력

4. **성별**
   - M: 남성
   - F: 여성

5. **연락처**
   - 형식: `010-XXXX-XXXX`
   - 하이픈 포함 입력
   - 예시: `010-1234-5678`

6. **담당 의사**
   - 드롭다운 메뉴에서 선택
   - 진료과별로 분류되어 표시됨

#### 선택 정보
7. **주소**
   - 환자의 거주지 주소

8. **이메일**
   - 선택 사항이지만 입력 권장
   - 진료 결과 및 알림 발송에 사용

9. **비상 연락처**
   - 형식: `010-XXXX-XXXX`
   - 보호자 또는 비상시 연락 가능한 번호

10. **보험 번호**
    - 건강보험증 번호

### Step 2: 정보 확인 및 등록

1. 모든 필수 정보 입력 확인
2. **"환자 등록"** 버튼 클릭
3. 입력 정보 최종 확인

### Step 3: 등록 완료 및 안내문 출력

등록이 완료되면 다음 정보가 포함된 팝업창이 표시됩니다:

```
환자 등록 완료

환자명: 홍길동
환자 ID: P202512060001
전화번호: 010-1234-5678

임시 비밀번호: Abc123!@
```

#### 중요 작업
1. **출력하기** 버튼 클릭하여 안내문 인쇄
2. 인쇄된 안내문을 환자에게 전달
3. 환자에게 다음 사항 안내:
   - 임시 비밀번호는 첫 로그인 시 변경 필요
   - 모바일 앱 다운로드 방법
   - 로그인 방법

#### 안내문에 포함되는 정보
- 환자 이름
- 환자 ID (로그인 시 사용)
- 전화번호
- 임시 비밀번호
- 앱 다운로드 QR 코드
- 로그인 방법 간단 안내

## 주의사항

### 개인정보 보호
1. ⚠️ **임시 비밀번호는 반드시 본인에게만 전달**
   - 다른 사람이 보지 않도록 주의
   - 전화나 이메일로 전달 금지

2. ⚠️ **주민등록번호 확인 철저**
   - 잘못된 주민번호 입력 시 등록 실패
   - 중복 주민번호 등록 불가

3. ⚠️ **화면에서 환자 정보 노출 주의**
   - 등록 완료 후 창 닫기
   - 타인이 볼 수 있는 공간에서 주의

### 입력 오류 방지
1. **연락처 형식 확인**
   - 하이픈 포함 필수
   - 올바른 형식: `010-1234-5678`

2. **담당 의사 선택 확인**
   - 진료과에 맞는 의사 선택
   - 잘못 선택 시 재등록 필요

3. **생년월일과 주민번호 일치 확인**
   - 시스템이 자동 검증하지만 수동 확인 권장

### 시스템 오류 발생 시
1. **오류 메시지 확인**
   - 빨간색 경고 메시지 내용 확인
   - 주로 입력 형식 오류 또는 중복 등록

2. **재시도**
   - 오류 수정 후 다시 등록 시도

3. **관리자 문의**
   - 반복적인 오류 발생 시 IT 담당자에게 문의
   - 오류 메시지 캡처하여 전달

## FAQ

### Q1: 환자가 임시 비밀번호를 잊어버렸어요
**A**: IT 관리자 또는 의사에게 요청하여 비밀번호 재설정이 필요합니다. 보안상 간호사가 직접 재설정할 수 없습니다.

### Q2: 주민번호 형식이 맞는데도 오류가 나요
**A**:
- 하이픈(-) 포함 여부 확인
- 총 13자리 확인
- 성별 코드(7번째 자리) 확인: 1,2,3,4 중 하나여야 함
- 이미 등록된 주민번호일 수 있으므로 환자 검색 먼저 수행

### Q3: 담당 의사가 목록에 없어요
**A**:
- 의사가 시스템에 등록되지 않았을 수 있습니다
- IT 관리자에게 의사 계정 생성 요청
- 임시로 다른 의사로 등록 후 나중에 변경 가능

### Q4: 등록 후 정보를 수정하려면?
**A**:
- 환자 목록에서 해당 환자 검색
- 환자 정보 수정 메뉴 이용
- 주민번호는 수정 불가 (재등록 필요)

### Q5: 외국인 환자는 어떻게 등록하나요?
**A**:
- 외국인 등록번호 입력 (주민번호 형식과 동일)
- 없을 경우 IT 관리자와 상담하여 대체 방안 마련

### Q6: 환자 등록 내역을 확인하려면?
**A**:
- 환자 목록 페이지에서 검색
- 이름, 전화번호, 환자 ID로 검색 가능

### Q7: 인쇄가 되지 않아요
**A**:
- 브라우저 인쇄 설정 확인
- 팝업 차단 해제 확인
- 프린터 연결 상태 확인
- 텍스트 복사 버튼으로 수동 복사 가능

### Q8: 같은 이름의 환자가 여러 명이에요
**A**:
- 생년월일로 구분
- 전화번호로 구분
- 등록 완료 후 생성되는 환자 ID로 명확히 구분 가능
