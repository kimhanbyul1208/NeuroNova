프롬7_12_01

React 프론트엔드를 개발하고 있다.
AI 추론 서버(Flask)는 다른 개발자가 담당한다.

AI 모델은 **30개의 단백질 값을 입력받아** 이를
**[정상, 감기, 독감, 코로나]** 4개 범주로 분류한다.

요구사항:

### 1) 단백질 3D 뷰 위치 변경
* 현재 단백질 3D 뷰가 AI 진단 영역에 들어가 있어 UI가 혼재되어 있다.
* 3D뷰는 **AI 진단과 완전히 분리**해서
  * 새 섹션 이름: **“단백질 3D뷰”**
  * AI 진단 카드/영역 옆으로 독립된 메뉴 또는 패널로 이동시켜 UI 논리를 명확히 한다.



### 2) 범주 배열 및 색상 규칙

* 범주의 정렬 순서는 **코로나 → 독감 → 감기 → 정상
* 각 범주는 **우선순위에 따라 색상을 명확하게 구분**해서 표현해야 한다.

  * 예:
    * 코로나: 가장 높은 경고 색
    * 독감: 그 다음
    * 감기: 중간 경고
    * 정상: 안정 색상
      (색상 스키마는 일관되고 직관적으로 구성)

### 3) 단백질 30개 시각화 버튼 UI
* 30개의 단백질 각각에 대해 **시각화 버튼을 생성**한다.

  * 버튼 클릭 시 해당 단백질의 상태/예측/그래프 등을 볼 수 있도록 한다.
* 버튼 디자인은 전체 React 스타일 가이드와 **미적 요소가 통일**되도록 수정한다.


### 4) 최종 목표

* React 전체 UI/UX 스타일을 일관성 있게 재정비
* AI 진단 섹션과 3D뷰 섹션을 명확히 구분
* 30개 단백질 버튼을 직관적·미적으로 구성
* 우선순위 기반 바이오마커 분류 색상 시스템을 적용
