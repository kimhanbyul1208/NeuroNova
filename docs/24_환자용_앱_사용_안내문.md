# NeuroNova 환자용 모바일 앱 사용 안내

## 환영합니다!

NeuroNova 모바일 앱을 이용해 주셔서 감사합니다.
본 앱을 통해 진료 예약, 진료 기록 조회, 복약 관리 등을 편리하게 이용하실 수 있습니다.

## 1. 앱 설치 방법

### Android
1. Google Play 스토어 실행
2. "NeuroNova" 검색
3. "설치" 버튼 클릭
4. 설치 완료 후 앱 실행

### iOS
1. App Store 실행
2. "NeuroNova" 검색
3. "다운로드" 버튼 클릭
4. 설치 완료 후 앱 실행

## 2. 최초 로그인 방법

### 준비물
병원에서 받은 안내문에 다음 정보가 기재되어 있습니다:
- **환자 ID**: 예) P202512060001
- **전화번호**: 예) 010-1234-5678
- **임시 비밀번호**: 예) Abc123!@

### 로그인 절차

#### 방법 1: 일반 로그인
1. 앱 실행
2. **"로그인"** 버튼 클릭
3. 환자 ID 입력 (병원에서 받은 ID)
4. 임시 비밀번호 입력
5. **"로그인"** 클릭

#### 방법 2: SMS 간편 로그인
1. 앱 실행
2. **"SMS 로그인"** 버튼 클릭
3. 병원에 등록한 전화번호 입력
4. SMS로 받은 인증번호 입력 (6자리)
5. **"확인"** 클릭

### 최초 로그인 시 비밀번호 변경 (필수)
보안을 위해 최초 로그인 시 반드시 비밀번호를 변경해야 합니다.

1. 로그인 후 자동으로 비밀번호 변경 화면 표시
2. **현재 비밀번호**: 임시 비밀번호 입력
3. **새 비밀번호**:
   - 8자 이상
   - 영문, 숫자, 특수문자 조합 필수
   - 예) MyPass123!@
4. **새 비밀번호 확인**: 동일하게 재입력
5. **"비밀번호 변경"** 버튼 클릭

⚠️ **주의**: 비밀번호는 타인에게 절대 알려주지 마세요!

## 3. 주요 기능 안내

### 3.1 홈 화면
로그인 후 첫 화면입니다.

#### 확인 가능한 정보
- 다가오는 예약 일정
- 최근 진료 내역
- 복용 중인 약 정보
- 새로운 알림

### 3.2 진료 예약
병원 방문 예약을 간편하게 신청할 수 있습니다.

#### 예약 방법
1. 하단 메뉴에서 **"예약"** 탭 선택
2. **"새 예약"** 버튼 클릭
3. 다음 정보 입력:
   - 진료과/담당 의사 선택
   - 희망 날짜 선택 (달력에서 가능한 날짜 표시)
   - 희망 시간대 선택
   - 증상 간단히 입력 (선택사항)
4. **"예약 신청"** 버튼 클릭
5. 예약 확정 시 푸시 알림 및 SMS 발송

#### 예약 확인 및 취소
- 예약 목록에서 상세 내용 확인 가능
- 예약일 24시간 전까지 취소 가능
- 취소는 해당 예약을 클릭 후 **"예약 취소"** 선택

### 3.3 진료 내역 조회
과거 진료 기록을 조회할 수 있습니다.

#### 조회 방법
1. 하단 메뉴에서 **"진료내역"** 탭 선택
2. 날짜별로 정리된 진료 기록 확인
3. 특정 진료 클릭 시 상세 정보 표시:
   - 진료 날짜
   - 담당 의사
   - 진단명
   - 처방약
   - 검사 결과

⚠️ **개인정보 보호**: 진료 기록은 본인만 확인 가능합니다.

### 3.4 복약 관리
처방받은 약의 복용 시간을 관리하고 알림을 받을 수 있습니다.

#### 기능
- 복용 중인 약 목록 확인
- 복용 시간 알림 설정
- 복용 완료 체크
- 약 복용 기록 조회

#### 알림 설정 방법
1. 진료내역에서 처방약 확인
2. 특정 약 선택
3. **"알림 설정"** 클릭
4. 복용 시간 설정 (아침, 점심, 저녁 등)
5. **"저장"** 클릭

### 3.5 알림
중요한 정보를 놓치지 않도록 알림을 받습니다.

#### 알림 종류
- 예약 확정 알림
- 예약 1일 전 리마인더
- 복약 시간 알림
- 검사 결과 업데이트 알림

#### 알림 확인
1. 하단 메뉴에서 **"알림"** 탭 선택
2. 최근 알림 목록 확인
3. 읽지 않은 알림은 파란색 점으로 표시

### 3.6 프로필 관리
내 정보를 확인하고 수정할 수 있습니다.

#### 수정 가능한 정보
- 연락처
- 이메일
- 주소
- 비상 연락처

#### 수정 방법
1. 하단 메뉴에서 **"프로필"** 탭 선택
2. **"정보 수정"** 버튼 클릭
3. 변경할 정보 입력
4. **"저장"** 클릭

⚠️ **수정 불가 정보**: 이름, 주민등록번호, 환자 ID는 병원에 직접 요청해야 합니다.

## 5. 자주 묻는 질문 (FAQ)

### Q1: 로그인이 안 돼요
**A**:
1. 환자 ID와 비밀번호 재확인
2. 병원에서 받은 안내문의 정보와 일치하는지 확인
3. 대소문자 구분에 유의
4. 임시 비밀번호를 변경했다면 새 비밀번호로 로그인
5. 계속 안 될 경우 병원에 문의

### Q2: 비밀번호를 잊어버렸어요
**A**:
- SMS 간편 로그인 이용
- 또는 병원 방문하여 비밀번호 재설정 요청

### Q3: 푸시 알림이 안 와요
**A**:
1. 휴대폰 설정 > 알림 > NeuroNova 앱 알림 허용 확인
2. 앱 내 알림 설정 확인
3. 휴대폰 절전 모드에서 앱 제외 설정

### Q4: 예약이 확정되었는지 모르겠어요
**A**:
- 예약 탭에서 상태 확인
  - **대기중**: 병원 확인 중
  - **확정**: 예약 완료
  - **취소**: 예약 취소됨
- 확정 시 푸시 알림 및 SMS 발송

### Q5: 앱이 느리거나 오류가 나요
**A**:
1. 앱 재시작
2. 휴대폰 재부팅
3. 앱 업데이트 확인 (Play Store/App Store)
4. 계속 문제 시 병원 IT 담당자에게 문의

### Q6: 진료 기록이 안 보여요
**A**:
- 최근 진료 후 기록 업로드까지 1~2일 소요
- 인터넷 연결 확인
- 앱 새로고침 (화면을 아래로 당기기)

### Q7: 다른 가족의 진료도 볼 수 있나요?
**A**:
- 현재는 본인 진료 기록만 조회 가능
- 향후 가족 계정 연동 기능 추가 예정

### Q8: 앱을 삭제하면 기록도 삭제되나요?
**A**:
- 기기 내 저장된 기록은 삭제됨
- 서버에는 계속 보관되므로 재설치 후 다시 조회 가능
